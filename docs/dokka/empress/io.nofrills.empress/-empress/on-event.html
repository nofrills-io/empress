onEvent

abstract fun onEvent(event: E, models: Models<M>, requests: RequestCommander<R>): Collection<M>

Handles an incoming event.

Parameters

event - An event that was triggered.

models - Current models.

requests - Allows to post new requests or cancel existing ones.

Return
A collection of updated models. You should only return models that have changed. If nothing has changed, you can return an empty collection. Based on updated models, a new update will be sent.
