enthrone

fun <Event, Patch : Any, Request> FragmentActivity.enthrone(empress: Empress<Event, Patch, Request>, id: String = DEFAULT_EMPRESS_ID, retainInstance: Boolean = true, dispatcher: CoroutineDispatcher = Dispatchers.Default): EmpressApi<Event, Patch>

Installs an empress instance into the activity. If you install more than one Empress into the same activity, you need to use different IDs for each instance.

Parameters

empress - Instance of Empress to install.

id - A unique ID that will be associated with the empress.

retainInstance - If true, the empress instance will be retained during configuration changes.

dispatcher - A dispatcher to use for empress.

Return
An instance of EmpressApi for communicating with empress.
